# Descrição do Projeto

O **Slime Runner** é um jogo no estilo *endless runner*, desenvolvido em **Python** utilizando a biblioteca **Pygame**.  
O jogador controla um slime que deve desviar de obstáculos, coletar power-ups, sobreviver pelo maior tempo possível e acumular pontuação progressiva.

Este projeto foi desenvolvido dentro da disciplina **Prática Profissional em ADS**, seguindo um processo iterativo.  
A **Iteração 2 (AC5)** trouxe melhorias significativas na jogabilidade, arquitetura, interface e organização do código.
